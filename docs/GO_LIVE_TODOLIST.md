# 上线 TodoList

## 0. 先确认当前状态

- [x] 前端 `npm run build` 已通过

## 1. 上线前必须处理的配置

### 后端

- [ ] 配置生产数据库连接串
  - 环境变量名：`ConnectionStrings__Default`
- [ ] 替换 JWT 密钥
  - 环境变量名：`Jwt__SecretKey`
  - 不能继续使用当前仓库里的默认值
- [ ] 确认 `Jwt__Issuer` 和 `Jwt__Audience`
- [ ] 确认邮件是否真的要启用
  - 如果启用：补齐 `Email__ApiKey` / `Email__FromAddress` 等配置
  - 如果暂时不用：保持 `Email__Enabled=false`
- [ ] 确认短信是否真的要启用
  - 如果不用：保持 `Sms__Enabled=false`
- [ ] 把所有生产密钥改放到部署平台环境变量
  - 不要继续写在 `appsettings.Development.json`
  - 不要把真实密钥提交到仓库

### 前端

- [ ] 配置生产 API 地址
  - 环境变量名：`VITE_API_BASE_URL`
  - 示例：`https://api.your-domain.com/api`
- [ ] 确认前端没有任何后端密钥

## 2. 上线前必须检查的代码行为

- [ ] 修改后端 CORS
  - 当前只允许 `http://localhost:5173`
  - 生产环境必须加入真实前端域名，否则登录和接口请求会被拦住
- [ ] 确认附件上传目录策略
  - 当前文件会保存到后端服务目录下的 `Uploads/`
  - 生产环境要确认这个目录会持久化、可备份、不会因重启或重新部署丢失
- [ ] 确认 HTTPS 正常工作
  - 后端已启用 `UseHttpsRedirection()`
- [ ] 确认数据库迁移可执行
  - 项目启动时会自动执行 `Database.MigrateAsync()`
